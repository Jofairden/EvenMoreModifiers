Detailed Description

This attribute is used to attach a certain Modifier to given ModifierEffects (can be attached to more than 1) This is used for detecting when effects need to become active (e.g. Light modifier is attached to the LightEffect, which makes LightEffect become active when an item with a +light modifier is active)
