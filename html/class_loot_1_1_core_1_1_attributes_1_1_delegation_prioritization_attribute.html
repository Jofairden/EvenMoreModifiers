Detailed Description

This attribute is used to set a custom prioritization for a delegation It allows you to customize at which point your delegation is called in the chain The end result is a prioritization list as follows: First part: all delegations prioritized as DelegationPrioritization.Early, order by their level Second part: all delegations with no custom prioritization Third part: all delegations prioritized as DelegationPrioritization.Late, order by their level To increase the force put into your prioritization, increase the delegation level
